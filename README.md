This script add modal to select year/month in Proton Calendar

### Instalation
- Add [tampermonkey](https://www.tampermonkey.net/index.php) in your web browser
- Add the [script.user.js](https://github.com/remigermain/proton-calendar-extra/raw/refs/heads/main/script.user.js) script in tampermonkey
- Go to [https://calendar.proton.me](https://calendar.proton.me)
